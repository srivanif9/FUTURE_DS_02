# FUTURE_DS_02
# 📊 Customer Retention & Churn Analysis

## 📌 Project Overview
The objective of this project is to analyze customer churn using the **IBM Telco Customer Churn Dataset**, identify the key factors affecting customer retention, and provide actionable business recommendations through data analysis and visualization.

## 🎯 Objectives
- Clean and preprocess the customer dataset.
- Analyze customer churn patterns.
- Identify factors influencing customer retention.
- Build meaningful visualizations.
- Create an interactive Power BI dashboard.
- Provide business recommendations to reduce customer churn.

## 📂 Dataset
**Dataset:** IBM Telco Customer Churn

**Total Records:** 7,043

**Features:** 21

**Target Variable:** Churn

The dataset contains customer demographic information, account details, subscribed services, billing information, and churn status.

## 🛠 Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Data Preprocessing

The following preprocessing steps were performed:

- Loaded the dataset
- Converted `TotalCharges` to numeric
- Handled missing values
- Removed unnecessary columns
- Checked duplicate records
- Performed exploratory data analysis (EDA)

## 📈 Visualizations

The project includes the following visualizations:

- Customer Churn Distribution
- Churn by Contract Type
- Churn by Internet Service
- Churn by Payment Method
- Monthly Charges vs Churn
- Tenure vs Churn
- Customer Retention Rate
- Correlation Heatmap

## 💡 Key Insights

- Customers with Month-to-Month contracts have the highest churn rate.
- Customers with shorter tenure are more likely to churn.
- Higher monthly charges are associated with increased customer churn.
- Customers with long-term contracts demonstrate better retention.

## ✅ Business Recommendations

- Encourage customers to switch to long-term contracts.
- Introduce loyalty rewards for existing customers.
- Offer discounts to high-risk customers.
- Improve customer engagement during the first year.
- Develop targeted retention campaigns.
